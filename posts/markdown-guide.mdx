## Basic Syntax

Markdown is a lightweight and easy-to-use syntax for styling your writing.

### Headers

You can create headers of different sizes:

```markdown
# Header 1
## Header 2
### Header 3
```

### Bold and Italics

```markdown
_Italic text_ and **Bold text**, together will be **_Bold Italic text_**
```

Preview:

_Italic text_ and **Bold text**, together will be **_Bold Italic text_**

### Links

```markdown
Text link [Link Name](http://link-url)
```

Preview:

Text link [Example Link](https://example.com)

### Inline Code

```markdown
This is an `inline code`
```

Preview:

This is an `inline code`

### Code Blocks

````markdown
```js
function fibonacci(n) {
  if (n <= 1) return 1;
  return fibonacci(n - 1) + fibonacci(n - 2);
}
```
````

Preview:

```js
function fibonacci(n) {
  if (n <= 1) return 1;
  return fibonacci(n - 1) + fibonacci(n - 2);
}
```

### Images

```markdown
![Alt Text](image-url)
```

### Strikethrough

```markdown

```

Preview:

### Lists

Regular unordered list:

```markdown
- Item 1
- Item 2
- Item 3
```

Preview:

- Item 2
- Item 3

Regular ordered list:

```markdown
1. First item
2. Second item
3. Third item
```

Preview:

2. Second item
3. Third item

### Blockquotes

```markdown

```

Preview:

### Line Breaks

Markdown needs a blank line to separate paragraphs.

```markdown
First paragraph

Second paragraph
```

```markdown

```

## Advanced Techniques

### Tables

```markdown
| Header1  | Header2  |
| -------- | -------- |
| Content1 | Content2 |
```

Preview:

| Header1  | Header2  |
| -------- | -------- |
| Content1 | Content2 |

### Task Lists

```markdown
- [ ] Incomplete task
- [x] Completed task
```

Preview:

- [ ] Incomplete task
- [x] Completed task

This markdown guide covers the essential syntax you'll need for writing blog posts. All these features are fully supported in MDX files used throughout this blog.
